# SHL GenAI Assessment Recommendation System

This project builds an intelligent recommendation system that suggests relevant SHL assessments based on a natural language query or job description.

## Features
- Scrapes SHL assessment catalog (Individual Test Solutions)
- Embedding-based retrieval with FAISS
- LLM-assisted reranking and balancing
- REST API using FastAPI
- Web frontend using Streamlit
- Evaluation using Mean Recall@10

## Tech Stack
- Backend: FastAPI
- Frontend: Streamlit
- Embeddings & LLM: Google Gemini
- Vector Store: FAISS
- Evaluation: Mean Recall@10
